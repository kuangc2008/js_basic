<h1>1 各种css属性<h1>

<pre>
color : 文本颜色
font-weight : 粗体
left : 制定左边位置
line-height : 行间距

top : 元素顶部位置
background-color : 背景颜色
border : 元素周围加边框
padding : 内编剧
font-size : 字体大小

text-align : 文本左对齐 居中  右对齐
letter-spacing : 字母之间的间距
font-style : 斜体文本
list-style : 改变选项的外观
background-image :  背景
</pre>
